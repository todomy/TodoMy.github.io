论国内“躺平”

Tip

最近“躺平”一词在国内热度比较高，我也写一下我对于“躺平”的看法，以下内容非AI，写的比较粗略。

什么叫做“躺平”

躺平以前是指打工上班的人，手头有积蓄，或者家里有矿，有条件，允许不上班，进行休息，不工作。

热度

国安部发布《煽动“躺平”的他们，正忙得脚不沾地》一文，引起了社会的关注，从国家角度去看，境外势力去资助国内躺平博主 ，宣扬躺平，这确实应该打击，国与国是竞争的，对方也在打击对方，双方有合作有斡旋，有博弈。

百姓如何去看

孔子曾言：‘’祸起萧墙‘’。近些年来国内随着房地产泡沫破碎，经济进行转型，还有海量的城投债务要进行化债，失业率高，即使不是失业，工作环境较差，996，内卷严重，以及拖欠工资等。娃卷学习，考大学，毕业后跑外卖，这种回报打破了原有的投资回报平衡，也能够说明国内经济有多差。大家都在为生计忍气吞声，然而突然有一篇文章说你们这样子算躺平了，不允许躺平，这里做个补充说明，国内大部分人是无法“躺平”的，中国目前还处于手停口停阶段，大部分百姓还是需要劳作才能温饱，租房，还房贷，养家糊口，这些都是需要用到钱的；所以引起了很大的民愤，百姓觉得他们不食烟火。

土地财政

土地财政，实际上是一个城市化发展很好的工具，低价购入土地，而且中国政府有强有力的行政手段，可以从农民手里征收土地，解决钉子户，积累出的财富在进行发展高新科技，国家基础设施建设。但问题土地财政关联的大头房地产，他实际上是一个缩量市场，人就那么多，房子的需求会越来越少，政府想要高价卖地，就要抬高房价，房地产商实际上大部分都是政府的白手套，或者是央企国企，他们也是需要盈利的，那么大家利益方向一致，就一起抬高房价，增加收益，早期城市周边居民因为政府有钱，赔偿给的高，有的赔了好几套房，然后房子又涨好几倍，导致财富直线上升，后期房地产不景气那么赔偿就少了，赔点钱甚至不够自己在原来位置买一套的。大批进城的农民农民工无社保无保障，参加这场城市化运动。农二代也进城开始租房，买房结婚。城市建设我有幸周围不少朋友干这块，有的开发商房子是豆腐渣工程，有的公交站台中标200万，层层外包，最后10万块。从这些可以看出这个经济发展模式比较粗犷，钱总是流向有权的利益方手里，百姓从劳作中赚取的酬劳最后还要去购买他们建设的豆腐在工程，资源的利用率实际上不是很高，发展模式并没有错，错在分配机制，错在如何高效透明，这些黑箱操作下，最终这个就会导致现在的情况，而且土地财政一旦政府依赖上了很多地方就靠卖地就行了，政府机关大家指望他发工资，领导靠这个收取好处。我知道的，老百姓也肯定是知道的，以前能够有口饭吃，现在由于吃口饭都不容易，最后还要被指责躺平，当然就比较生气。
土地财政是个好东西，但是如何用才是关键

百姓共识

  1. 动物知道环境差，减少繁衍的后代，中国老百姓虽然没有反抗力量，但是从生育可以直接看出来，这种下跌速度是历史之最，经济如果是好的，还能够维持生育率，目前经济如此艰难，中国人口很难再像建国后行政动员一家生四五个，其次后续又伴随着计划生育。所有的强制行政命令执行干预人口都会迎来反噬，导致人口结构太差。
  2. 现在工作如此的卷，赚的钱还一点点，最后身体差了还要花更多的钱去看病，身体还受罪，还不如一开始就以身体为主。
  3. 工资低，彩礼高，房子贵，就业不稳定，不婚不育保平安
  4. 经济压力大导致男女对立，结婚必买房更是打消了不少人结婚生子的想法
    看起来百姓最后都是无奈的选择，被迫的事情就会导致有怨气，有意识形态

执政者角度

  1. 目前情况保就业，但是经济周期客观规律并不是一纸命令就能够搞定的，需要时间
  2. 土地财政带来的房地产债务，城投债务需要化债，通过金融工具缓慢的让全社会承担
  3. 稳定金融
  4. 加速转型
  5. 加强舆论管控，增加维稳费用
    政权的稳定才是第一位

个人角度

首先要认清自己在国内的身份，举例如果你是汉人，你的地位比少数民族还要低，但是这个国家你要知道属于是汉人的国家，百分之90的人口是汉人，其次执政里面顶层也都是汉人，你只是不在统战价值而已，当然也不如港澳台这些华侨地位高，如何发展：

  1. 优先去江浙沪，珠三角，去就业，这些地方政府相对开明，经济更强，在大的市场里参与建设才能有回报
  2. 去不了上面的也要去各自的省会，人口的下跌大部分地区将会没有发展的机会，有人口才会有发展
  3. 就业选择行业避开跟房地产相关行业，不要着急赚大钱，先熟悉自己能力，多尝试选择合适自己的战场
  4. 媒体舆论看看就好，互联网媒体都是吃人血馒头的，不要在意，不要被媒体牵着走
  5. 培养自己在规则内博弈的能力，保持头脑清醒，不要被宏大叙事，其他人影响你个人决策，脑子永远属于自己
  6. 当你经济到了一定的层次一定要去想办法搞国外绿卡，香港身份，有了这些你在国内不管是投资还是孩子念书都轻松，属于是人上人了，有政策优待
  7. 资产一定要全球化，不要放到一个篮子里，一定要优先将资金放在国家政权稳定，法律法规严格，尊重个人财富的地方。
  8. 个人要建立自己是地球人，全球化思维，而不是局限于一国，各个国家都会发展本国的爱国主义，平时宣传上也都是偏向自己，做人一定要兼听则明，尽量多想想地球，少一些美国人太坏了，国家对你怎么样，你就对国家怎么样，别人对你怎么样，你就对他怎么样就行了，简单明了
  9. 环境不好，孟母还知道三迁，你自己在国内实力不行，倒霉的就是你，国内规则如此，想要改变就去学习就去锻炼自己，适应环境，活下去，国家也希望你自强
  10. 这个世界永远是：制定规则> 规则赢家 >规则输家 ，你需要根据你所处环境去判断，自己环境是属于弱肉强食还是普通人也有尊严的活下去的地方，根据情况去制定你的生存策略
  11. 你拥有在地球迁移的权利，前提是你有能力去改变，改变不了就要接受，至少这样子不会抑郁，身体健康永远是第一
