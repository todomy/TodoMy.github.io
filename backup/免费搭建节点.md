#  科研党必备：低成本搭建专属科研资料查询节点实操指南

对于许多身处大陆、偶尔有查询海外科研资料需求的学者和学生来说，购买昂贵的长期服务往往“性价比不高”，而申请正规网络线路流程又过于繁琐。

本文将介绍一种利用**云服务器**与**开源镜像**结合的搭建方法。通过此方案，你可以获得一个基于 VLESS + WebSocket + TLS 加密的代理节点，既能满足科研查询需求，又能保证较高的性价比。

---

## ️ 核心原理与技术架构

本方案的核心在于利用 **Docker 镜像 (`7techlife/seven`)** 进行快速部署。

*   **底层技术**：基于 `sing-box` 构建，采用 **VLESS + WebSocket** 协议。
*   **安全隧道**：通过 **Cloudflare Tunnel** 暴露服务，无需公网 IP，且流量经过 TLS 加密（由 Cloudflare 提供）。
*   **流量伪装**：WebSocket 流量承载于 TLS 之上，看起来像普通的 HTTPS 流量，具有较强的隐蔽性。

##  详细操作步骤

请按照以下 5 个步骤进行操作：

### 第一步：注册与登录
1.  访问爪云注册地址：[https://run.claw.cloud/](https://run.claw.cloud/)
2.  如果你拥有 Google 账号或 GitHub 账号，每月可获得 **5美元** 的额度，这对于搭建节点非常划算。
3.  选择授权登录（Google 或 GitHub），完成账号绑定。

### 第二步：创建应用
1.  进入桌面 App **Launchpad**，点击右上角的 **Create App**。
2.  在 **Image Name**（镜像名称）一栏中，填写开源镜像地址：`7techlife/seven`。
3.  **地区选择**：建议选择 **日本**。因为日本物理距离中国大陆最近，通常延迟最低，访问速度最快。
4.  其他选项保持默认，点击保存。

### 第三步：获取节点信息
1.  创建完成后，点击进入你的项目。
2.  在右下角找到 **Logs**（日志）选项卡。
3.  你将看到生成的订阅节点信息。
4.  使用支持 VLESS (V2ray) 协议的客户端，直接**订阅**该链接即可。

<img width="1347" height="882" alt="Image" src="https://github.com/user-attachments/assets/1c8d6511-2521-4cdf-9a60-e76f964d37f1" />

### 第四步：客户端配置
*   复制生成的 VLESS 链接（或聚合链接），在你的客户端中尝试“从剪贴板导入”。
*   **注意**：VLESS 链接中使用了常见的网站作为“伪装域名”，但实际连接会通过 SNI 和 Host 指向你的隧道域名。

### 第五步：节点维护（防失效）
*   如果节点失效，只需在云平台**重启**该项目。
*   重启后，Logs 会自动更新生成新的节点信息，你只需重新订阅即可，无需重新部署。

##  进阶技巧：两种运行模式

根据你的需求，该镜像支持两种运行模式（基于环境变量控制）：

### 1. 临时模式 (最简单)
*   无需任何额外配置。
*   脚本自动生成 UUID 和临时的 `*.trycloudflare.com` 隧道域名。
*   **缺点**：容器重启后节点信息会改变，适合短期测试。

### 2. 固定模式 (推荐)
*   需要在运行时提供 `uuid`、`token` 和 `domain` 三个环境变量。
*   **优点**：节点信息在容器重启后保持不变，稳定性更高。
*   *注：此模式需要你在 Cloudflare Zero Trust Dashboard 中预先配置隧道，适合有长期使用需求的用户。*
